# XADE Development Guidelines

## Git Workflow

### Branch Naming
- `feature/short-description` - New features
- `fix/short-description` - Bug fixes
- `docs/short-description` - Documentation updates
- `refactor/short-description` - Code refacotring

### Commit Messages:
Follow conventional commits:

feat(desktop): add heatmap visualization component
fix(api): handle missing image metadata gracefully
docs(api): update analyze endpoint response schema
